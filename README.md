# Car Prices ETL & Analytics

A Data Engineering project that cleans a dataset of 500k+ used car auction records using Python (Pandas) for visualization in Power BI.

## Project Structure
- `data/raw/`: Contains the original `car_prices.csv`.
- `data/processed/`: Contains the cleaned output.
- `scripts/`: Python ETL logic.
- `requirements.txt`: Project dependencies.
